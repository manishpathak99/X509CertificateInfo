# Android X509 Certificate Info
An Android application which shows X509 certificate information on screen. 
  The information includes
  
  
  1- algorithm
  
  2- Size  
  
3- Cert version

4- issuerdn

5- subjectdn

6- serial

7- created time

8- expiry time

9- MD5, SHA-1, SHA256
